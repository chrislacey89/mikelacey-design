# Kudos Specification

## Overview
A gallery showcasing photos of "attaboys" — actual thank-you notes, letters, and documents from colleagues and industry professionals. Photos are displayed in a grid with lightbox viewing. Ends with a CTA to get in touch.

## User Flows
- Visitor browses the grid of attaboy photos
- Visitor clicks a photo to view it enlarged (lightbox)
- Visitor clicks "Get in Touch" CTA → navigates to Connect section

## UI Requirements
- Photos displayed in a masonry or grid layout
- Lightbox on click to view document photos enlarged
- "Get in Touch" CTA button at the end
- Responsive layout for mobile

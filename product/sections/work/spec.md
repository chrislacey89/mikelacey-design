# Work Specification

## Overview
A portfolio page showcasing Mike's career through three stacked sections: a list of production credits (show name and network), a photo gallery with lightbox viewing, and two embedded YouTube video interviews. Ends with a CTA to get in touch.

## User Flows
- Visitor scrolls through credits to see the breadth of Mike's work
- Visitor browses the photo gallery grid and clicks a photo to view it enlarged (lightbox)
- Visitor watches embedded YouTube video interviews
- Visitor clicks "Get in Touch" CTA → navigates to Connect section

## UI Requirements
- Credits displayed as a single list with show/project name and network
- Photo gallery in a grid layout with lightbox on click
- Two YouTube video embeds
- Stacked sections: Credits → Gallery → Videos → CTA
- "Get in Touch" CTA button at the end
- Responsive layout for mobile
